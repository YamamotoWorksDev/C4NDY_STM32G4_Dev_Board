# C4NDY_STM32G4_Dev_Board
STM32G491CET6を搭載した開発ボード

## specifications
- STM32G491CET6(Arm Cortex-M4/170MHz max)
- External Crystal Oscilator(12.288MHz)
- USB Fullspeed
- USB Bus power & Self power(max 5.5V)
